# Glentech YouTube Downloader

Futuristic YouTube downloader with search, MP3/MP4 quality selector, and backend proxy.

## Setup

1. Clone repo
2. `npm install`
3. `node server.js`
4. Open `http://localhost:3000`

## Features
- Search YouTube
- Paste YouTube link
- MP3: 64/128/320 kbps
- MP4: 360/720/1080p
- Dark neon futuristic Glentech style
- Backend proxy hides API key
